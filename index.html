Using

Welcome to ProtoLife!

How?

1. Select a color with your keypad

2. Click on the grid with the mouse

3. Click 'd' and then click on block's to delete

3. Press start and watch!

What?

Each block behaves according to a set of rules that can be specified. Click on the colored squares to specify the rules of a block. Read the documentation to see what rules are allowed.
